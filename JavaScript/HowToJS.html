Reverse the string : Show Ans:
			var dwarves = "bashful doc dopey grumpy happy sleepy sneezy";
			Output : "sneezy sleepy happy grumpy dopey doc bashful"
			


				Single line of JS code has achieved this :
					var strReverse = dwarves.split(" ").reverse().join(" ");
				Code Explanation : 
					split function creates an array of string separated by " ".
					reverse does the obvious i.e. revered word array.
					join creates a single string using array elements separated by " ".
					FYI - if you miss " " and instead specify "", the string would reverse by character.
